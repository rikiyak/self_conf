# self conf

## vim

### install
```
brew install vim
```

### setup
```
cp ~/.vimrc ~/.vimrc.org
cp vim/vimrc ~/.vimrc
```
## zsh

### install
[prezto](https://github.com/sorin-ionescu/prezto)

### setup
```
cp zsh/preztorc ~/.preztorc
cp zsh/zshrc ~/.zshrc
```
## tmux

### install
```
brew install tmux
```

### setup
```
cp ~/.tmux.conf ~/.tmux.conf.org
cp tmux/tmux.conf ~/.tmux.conf
```
